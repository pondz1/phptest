Nami

ชื่อ : นามิ (Nami)

ฉายา : แมวโขมยนามิ

ผลปีศาจ : ไม่มี

ตำแหน่ง : ต้นหน

ค่าหัวล่าสุด : 66 ล้านเบรี

ความฝัน : เขียนแผนที่โลกขึ้นด้วยตนเอง

ความสูง : 169 cm

อายุ : 20 ปี

วันเกิด : 3 กรกฏาคม

นามิเป็นลูกบุญธรรมของ เบลเมล ซึ่งก็มี นามิ และก็ โนจิโกะ ทั้ง 3 นั้นได้อยู่กันเป็นเหมือนครอบครัว แต่แล้ววันหนึ่ง กลุ่มโจรสลัดอารองก็มายึดเกาะ โคโคยาชิ ที่ทั้ง 3 อยู่นั่นเอง ซึ่ง อารองก็ให้ทุกคนในเกาะ โคโคยาชิ ต้องจ่ายเงินภาษีให้อารอง เพื่อที่ทุกคนจะได้มีชีวิตอยู่ต่อไป เบลเมลนั้นซึ่งไม่สามารถจ่ายภาษีไหว ซึ่งเหตุการณ์มันเกิดขึ้นเนื่องจากกลุ่มอารองไปเจอ ลูกของเบลเมลเข้านั่นเอง เนื่องจากที่เบลเมลนั้นไม่สามารถบอกว่า นามิ และ โนจิโกะ ไม่ใช่ลูกของเธอเพราะเธอสนิทสนมกับทั้ง 2 จนเป็นเหมือนครอบครัวเดียวกันไปแล้ว
